## 프로젝트 기획 - 개발 범위 소개
- **회원 관리 기능** - 회원 가입, 로그인 및 로그아웃

     **(회원가입시 주소Api, 카카오 소셜로그인 구현)**

- **게시판 관리 기능** - 카테고리 별 조회, 페이징, 파일 첨부, 게시글 등록, 수정

     **(카카오톡 공유 기능 구현)**

- **기부 및 응원하기 기능** - 기부 금액 증가, 응원 수 증가

- **회원 활동 내역 관리 기능** - 마이페이지(나의글, 내 기부)
